和声
====

该文档涵盖了音程的相关知识.

阅读完该文档后，您将会了解到:

* 和声的概念.
* 和声的种类.

--------------------------------------------------------------------------------

和声的概念
----------
三个音或者三个以上的按照特殊关系排列, 同时发出或者顺次发出的声音.

和声的种类
----------
- 三和弦
- 二和弦
- 四和弦
- 六和弦
- 七和弦
- 九和弦
- 十一和弦
- 十三和弦

### 三和弦
可以简单地认为三和弦由三个音组成

```
根音 三度 五度
```

三和弦可以分为

- 大三和弦 大三度 + 小三度
- 小三和弦 小三度 + 大三度
- 增三和弦 大三度 + 大三度
- 减三和弦 小三度 + 小三度
